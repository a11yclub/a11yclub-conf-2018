#### Keyboard shortcuts

* `Page Up`, `←`, `↑`: Previous slide
* `Space`, `Return`, `Page Down`, `→`, `↓`: Next slide
* `Ctrl`: Show / hide mouse cursor
* `Shift`: Enter / leave interactive mode (stepwise display of list items)
* `Esc`: Slide overview
